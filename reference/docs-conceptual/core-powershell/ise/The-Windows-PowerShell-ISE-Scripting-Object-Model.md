# <a name="the-windows-powershell-ise-scripting-object-model"></a>O ISE do Windows PowerShell modelo de objeto de Scripting
  O Windows PowerShell® Integrated Scripting Environment (ISE) expõe a funcionalidade através de um modelo de objeto de scripting subjacente que permite aos utilizadores escrever scripts Windows PowerShell que manipular os diferentes aspetos visual e funcionais do ISE do Windows PowerShell. Objetos de scripts, as respetivas propriedades e os respetivos métodos são definidos especificamente para esta finalidade.
